# Coralluna Theme

A deep soft dark-blue VSCode theme with coral forest colors, designed to be friendly for Godot game development.

![Coralluna Theme Preview](https://via.placeholder.com/800x450/1a1a2e/16213e?text=Coralluna+Theme+Preview)

## Installation

### Option 1: Install from VSIX (Recommended for Development)

1. Download the latest `.vsix` file from the [releases page](https://github.com/your-repo/coralluna-theme/releases)
2. In VSCode, go to **Extensions** (`Ctrl+Shift+X` or `Cmd+Shift+X`)
3. Click the `...` menu in the top-right corner
4. Select **Install from VSIX**
5. Navigate to the downloaded `.vsix` file and select it

### Option 2: Install from VSCode Marketplace

1. Open VSCode
2. Go to **Extensions** (`Ctrl+Shift+X` or `Cmd+Shift+X`)
3. Search for "Coralluna"
4. Click **Install**

## Color Palette & Design Philosophy

Coralluna is inspired by the serene beauty of coral forests deep in the ocean — a peaceful, deep blue environment where vibrant coral formations create a calming yet engaging coding experience.

### Primary Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Deep Ocean | `#1a1a2e` | Editor background |
| Lavender Mist | `#b6a3ff` | Variables, tags |
| Coral Pink | `#ff6bb0` | Keywords, control flow |
| Seafoam | `#19f9d8` | Classes, types, functions |
| Azure | `#4adbff` | Methods, function calls |
| Salmon | `#ffa38c` | Numbers, constants, parameters |
| Warm Sand | `#ffe4a1` | Strings |
| Soft Sky | `#87b3ff` | Operators, enum members |
| Pale Mist | `#c8d5e6` | Default text, punctuation |

### Design Principles

1. **Soft Contrast**: Reduced eye strain with carefully balanced contrast ratios
2. **Godot-Friendly**: Special syntax highlighting for GDScript and Godot-specific patterns
3. **Language Support**: Optimized token colors for 20+ programming languages
4. **Semantic Highlighting**: Enhanced semantic tokens for better code readability
5. **Consistency**: Unified color coding across all language syntax elements

### Supported Languages

- **Game Development**: GDScript, C#, HLSL, Shader languages
- **Web**: JavaScript, TypeScript, HTML, CSS, SCSS, JSON, Markdown
- **Systems**: Rust, C, C++, Go
- **Scripting**: Python, PHP, Ruby, Lua
- **Functional**: Haskell, Elixir, F#, OCaml
- **Data**: YAML, TOML, XML

## Activation

1. Open VSCode
2. Go to **Preferences** → **Color Theme** (or press `Ctrl+K Ctrl+T` / `Cmd+K Cmd+T`)
3. Select **Coralluna** from the list

The theme will be immediately applied to your editor.
